# קופת טוב וחסד - מערכת ניהול נתמכים

מערכת מקיפה לניהול משפחות נתמכות, בקשות תמיכה, תמיכות ופרויקטים.

## 🚀 התקנה והרצה

### דרישות מקדימות
- Node.js 18+
- חשבון Supabase (חינם)

### שלב 1: התקנת תלויות
```bash
npm install
```

### שלב 2: הגדרת משתני סביבה
צור קובץ `.env` בתיקייה הראשית:
```
VITE_SUPABASE_URL=https://odmxtufodaljukdhxggs.supabase.co
VITE_SUPABASE_ANON_KEY=your-anon-key-here
```

### שלב 3: יצירת טבלאות במסד הנתונים
1. פתח את Supabase Dashboard
2. לך ל-SQL Editor
3. העתק את התוכן מ-`supabase/schema.sql`
4. הרץ את הקוד

### שלב 4: הפעלת המערכת
```bash
npm run dev
```

המערכת תעלה בכתובת: http://localhost:5173

## 📁 מבנה הפרויקט

```
├── src/
│   ├── components/     # קומפוננטות משותפות
│   ├── pages/          # דפי האפליקציה
│   ├── lib/            # ספריות ו-types
│   └── index.css       # עיצוב גלובלי
├── supabase/

```
